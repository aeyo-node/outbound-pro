EV  Charging  Troubleshooting  
Instructions
 
Document
 
Introduction  
This  document  contains  standard  troubleshooting  procedures  for  common  EV  
charging-related
 
issues.
 
These
 
instructions
 
are
 
intended
 
to
 
help
 
support
 
teams
 
guide
 
customers
 
effectively
 
and
 
escalate
 
issues
 
whenever
 
required.
 
 
1.  Manual  Release  Playbook  
Issue  Scenario  
Gun  stuck  and  unable  to  release  the  charging  cable.  
Instructions  
1.  Stop  the  charging  session  from  the  ChargeMOD  app.  2.  Use  the  vehicle  key  to  lock  and  unlock  the  vehicle  3  times,  then  try  removing  the  gun.  3.  If  the  gun  is  still  not  released,  press  the  unlock  button  on  the  vehicle  dashboard  2  
times
 
and
 
try
 
removing
 
the
 
gun
 
again.
 4.  Turn  the  vehicle  ignition  ON  and  try  removing  the  gun.  
 
(This
 
step
 
should
 
only
 
be
 
tried
 
for
 
Tata
 
vehicles.)
 5.  If  the  issue  still  persists,  apply  the  brake  and  engage  the  hand  brake.  
 
Then
 
turn
 
the
 
vehicle
 
ignition
 
ON
 
and
 
wait
 
for
 
10
 
seconds.
 
 
After
 
that,
 
turn
 
the
 
ignition
 
OFF
 
and
 
release
 
the
 
hand
 
brake.
 
 
(Release
 
the
 
brake
 
only
 
if
 
the
 
vehicle
 
will
 
not
 
move.)
 
 
Then
 
try
 
removing
 
the
 
gun
 
again.
 
 
(This
 
step
 
should
 
only
 
be
 
tried
 
for
 
Tata
 
vehicles.)
 6.  If  no  other  charging  sessions  are  running  at  the  station,  restart  the  charger  by  turning  
the
 
machine
 
power
 
OFF
 
and
 
ON.
 7.  Alternatively,  after  restarting  the  machine,  when  the  gun  status  changes  to  
“Preparing,”
 
start
 
and
 
stop
 
the
 
charging
 
session
 
once
 
again,
 
then
 
unlock
 
and
 
try
 
removing
 
the
 
gun.
 8.  If  the  issue  is  still  not  resolved,  explain  the  manual  release  option  to  the  customer  or  
share
 
the
 
manual
 
release
 
video/instructions.

2.  Handshake  Timeout  Instructions  
Issue  Scenario  
Preparing  state  for  more  than  2  minutes  OR  Preparing  →  Finished  without  entering  Charging  
state.
 
Instructions  
1.  Confirm  whether  the  vehicle  ignition  is  OFF,  the  brake  is  applied,  the  hand  brake  is  
engaged,
 
and
 
the
 
vehicle
 
is
 
locked
 
using
 
the
 
key.
 2.  Remove  the  gun  from  the  vehicle  and  reconnect  it  properly  once  again.  3.  Press  and  hold  the  gun  firmly  into  the  vehicle  charging  port  and  initiate  the  charging  
session
 
within
 
40
 
seconds.
 4.  If  another  connector  is  available,  ask  the  user  to  try  charging  using  the  other  
connector.
 5.  If  the  same  issue  continues,  ask  the  user  to  move  the  vehicle  slightly  forward  or  
backward
 
and
 
try
 
charging
 
again.
 6.  If  the  charger  supports  reset,  perform  a  charger  reset.  Otherwise,  turn  the  machine  
OFF
 
and
 
ON,
 
then
 
try
 
charging
 
again.
 7.  If  the  issue  repeats,  ask  the  user  to  try  another  connector  on  the  same  charger  or  a  
nearby
 
charger.
 8.  If  the  issue  is  still  not  resolved,  ask  the  user  to  try  another  station.  Inform  the  
technical
 
team
 
to
 
monitor
 
the
 
charger.
 
 
3.  Low  SoC  Charging  Instructions  
Issue  Scenario  
Vehicle  refusing  DC  fast  charging  due  to  very  low  State  of  Charge  (SoC  below  15%).  
Instructions  
1.  Ask  the  user  to  move  the  vehicle  slightly  forward  and  backward,  then  try  charging  
again.
 2.  Press  and  hold  the  gun  firmly  into  the  vehicle  charging  port  and  initiate  the  charging  
session
 
within
 
40
 
seconds.
 3.  If  another  connector  is  available  and  free  on  the  charger,  ask  the  user  to  try  charging  
using
 
the
 
other
 
connector.
 4.  If  the  charger  supports  reset,  perform  a  clear  cache  and  reset  operation.  Otherwise,  
turn
 
the
 
machine
 
OFF
 
and
 
ON,
 
then
 
try
 
charging
 
again.
 
--- PAGE 3 ---
5.  If  the  vehicle  is  still  not  charging,  ask  the  user  to  increase  the  battery  level  by  around  
10%
 
using
 
an
 
AC
 
charger
 
and
 
then
 
retry
 
DC
 
charging.
 6.  If  AC  charging  is  not  available,  suggest  using  a  portable  charger  with  a  3-pin  socket  if  
available.
 
 
4.  Loose  Gun  /  Loose  Connector  
Instructions
 
Issue  Scenario  
Charger  status  continuously  changing  between  Available  and  Preparing  without  reaching  
Charging
 
state.
 
Instructions  
1.  During  charging  initiation,  try  removing  the  gun  once.  2.  If  the  gun  can  be  removed  easily,  inform  the  customer  that  the  gun  is  not  locked  
properly
 
and
 
the
 
issue
 
is
 
due
 
to
 
a
 
loose
 
connection.
 3.  Reconnect  the  cable  properly  and  ensure  that  the  vehicle  is  locked.  4.  Press  and  hold  the  gun  firmly  into  the  vehicle  charging  port  and  initiate  the  charging  
session
 
within
 
40
 
seconds.
 5.  If  another  connector  is  free  and  available  on  the  machine,  ask  the  user  to  try  
charging
 
using
 
the
 
other
 
connector.
 6.  Ask  the  user  to  move  the  vehicle  slightly  forward  and  backward,  then  try  charging  
again.
 7.  If  the  machine  supports  reset,  perform  a  clear  cache  and  reset  operation.  Otherwise,  
turn
 
the
 
machine
 
OFF
 
and
 
ON,
 
then
 
try
 
charging
 
again.
 8.  If  a  manual  lock  option  is  available,  try  using  it.  Otherwise,  share  the  related  manual  
lock
 
video/instructions
 
with
 
the
 
customer.
 9.  If  the  issue  is  still  not  resolved,  ask  the  user  to  try  AC  charging  first  and  then  retry  DC  
charging.
 
If
 
the
 
issue
 
still
 
continues,
 
suggest
 
another
 
available
 
station.
 10.  If  the  same  issue  has  been  reported  by  other  customers  on  this  machine,  report  it  to  
the
 
technical
 
team.
 
 
5.  Supply  Side  Issue  /  Voltage  Issue  
Instructions
 
--- PAGE 4 ---
Issue  Scenario  
Machine  fault  showing  OR  charging  stopping  due  to  a  supply  side  issue.  
Instructions  
1.  Check  the  machine  display.  Confirm  whether  the  machine  display  is  ON,  the  network  
is
 
connected,
 
and
 
whether
 
any
 
error
 
message
 
is
 
being
 
displayed.
 2.  If  everything  appears  normal,  check  the  panel  box  and  confirm  whether  3-phase  
supply
 
is
 
available
 
and
 
the
 
voltage
 
is
 
proper.
 3.  If  there  is  a  voltage  issue  or  supply  is  missing  in  any  phase,  inform  the  customer  that  
it
 
is
 
a
 
supply
 
side
 
issue
 
and
 
charging
 
can
 
resume
 
only
 
after
 
the
 
power
 
supply
 
becomes
 
stable.
 4.  If  required,  suggest  a  nearby  available  charging  station  to  the  customer.  
 
6.  Charger  Disconnect  /  Network  Issue  
Instructions
 
Issue  Scenario  
Charger  showing  “Disconnected”  during  the  charging  session.  
 
A.  Charger  Showing  Disconnected  
Instructions  
1.  Check  whether  the  machine  display  is  ON.  2.  If  other  chargers  are  available  at  the  station,  check  whether  all  chargers  are  
disconnected.
 
 
Also
 
confirm
 
whether
 
all
 
chargers
 
got
 
disconnected
 
at
 
the
 
same
 
time.
 3.  Check  whether  supply  is  available  in  all  3  phases.  4.  Check  whether  the  network  is  connected  on  the  machine  display  and  whether  the  
network
 
symbol
 
is
 
showing
 
properly.
 5.  If  the  network  is  not  connected,  turn  the  machine  power  OFF  from  the  panel  box,  wait  
for
 
10
 
seconds,
 
and
 
turn
 
it
 
ON
 
again.
 6.  If  the  network  does  not  reconnect  even  after  waiting  for  10  minutes,  guide  the  
customer
 
to
 
another
 
available
 
station.
 7.  Report  the  issue  to  the  technical  team.  
--- PAGE 5 ---
B.  Network  Disconnect  During  Charging  
Instructions  
1.  Check  the  last  log.  If  the  current  time  log  is  not  available,  confirm  that  the  charger  is  
disconnected.
 2.  Inform  the  customer  that  the  issue  is  due  to  a  network  disconnection.  3.  If  the  charging  needs  to  be  stopped,  ask  the  customer  to  press  the  emergency  button  
and
 
turn
 
it
 
to
 
the
 
right
 
to
 
release
 
it,
 
then
 
remove
 
the
 
gun.
 
 
(Perform
 
this
 
step
 
only
 
if
 
no
 
other
 
vehicles
 
are
 
charging
 
on
 
the
 
charger.)
 4.  If  that  does  not  work,  turn  the  machine  power  OFF  from  the  panel  box  to  stop  
charging.
 
 
(Perform
 
this
 
step
 
only
 
if
 
no
 
other
 
vehicles
 
are
 
charging
 
on
 
the
 
charger,
 
or
 
only
 
after
 
getting
 
permission
 
from
 
customers
 
whose
 
charging
 
sessions
 
are
 
active.)
 
 
C.  Machine  Trip  Issue  
Instructions  
1.  Confirm  whether  the  machine  is  completely  OFF.  2.  Check  whether  supply  is  available  in  all  3  phases.  3.  If  supply  is  available,  check  the  panel  box.  
 
If
 
it
 
can
 
be
 
opened,
 
check
 
whether
 
any
 
breaker
 
has
 
tripped.
 4.  If  a  breaker  is  tripped,  turn  it  ON  and  check  whether  the  machine  powers  ON  
properly.
 5.  If  no  external  breaker  issue  is  found,  inform  that  the  trip  may  be  from  inside  the  
machine
 
and
 
a
 
technician
 
needs
 
to
 
inspect
 
it.
 6.  Report  the  issue  to  the  concerned  person  /  technical  team.
